About

A group of students with passion to learn and develop. We learn by making and we enjoy making different fun and awesome projects. The point is to learn and develop skills while making friends and enjoying the journey. We want to create a space where we can all discuss and work on skills like ioT, Web Dev, AI, Machine Learning, Robotics, Photography and more (There are other secret sects like Anime, Piracy, etc 🤫🤫).
